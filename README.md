# Meine LeetCode-Lösungen

Hallo! Hier speichere ich meine Lösungen für Aufgaben von [LeetCode](https://leetcode.com).

Hier ist Link zu meinem Profil: https://leetcode.com/u/CrashSystem/

## 📂 Wo sind die Lösungen?
Alle Codes findest du im Ordner `my-solutions/`. 
Jede Aufgabe hat einen eigenen Ordner mit meinem Code und dem Text der Aufgabe.

## 🤖 Wie funktioniert das?
Ich muss den Code nicht selbst hochladen. Ein automatischer Bot (GitHub Actions) kopiert meine Lösungen von LeetCode direkt hierher, wenn ich eine Aufgabe richtig gelöst habe.
